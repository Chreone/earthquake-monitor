About this analysis
This tool analyzes historical earthquake patterns to identify areas with similar seismic behavior to past events. This is a statistical pattern analysis, not a prediction. Earthquake prediction is scientifically impossible. Always follow PHIVOLCS for official information.
ABOUT EQWATCH

EQWatch is a seismic pattern analysis platform for Mindanao, Philippines, providing earthquake monitoring and statistical pattern identification based on historical PHIVOLCS data.

Important Disclaimer
Earthquake prediction is scientifically impossible. This tool shows statistical patterns based on historical data only. It cannot and does not predict future earthquakes. Always follow official PHIVOLCS and NDRRMC advisories.

Data Source
All earthquake data is sourced from PHIVOLCS (Philippine Institute of Volcanology and Seismology), covering records from 2016 to present.

Methodology
The KNN machine learning model identifies grid cells with similar seismic patterns to historically active areas. This is a retrospective pattern analysis, not a forward-looking prediction.
